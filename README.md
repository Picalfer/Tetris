# Mega Tetris

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)

Классическая игра Тетрис для Android с собственным дизайном и полной игровой механикой.

## О проекте

Полнофункциональная игра Тетрис, созданная при изучении Kotlin и Android разработки. Включает все основные возможности классической игры.

**Основной функционал:**
- Полная игровая механика Тетрис
- Система подсчета очков и рекордов
- Кастомная графика и анимации
- Управление свайпами
- Сохранение лучшего результата

## Технологии

* Kotlin
* Custom View
* ViewBinding
* SharedPreferences
* Canvas
* Game logic
